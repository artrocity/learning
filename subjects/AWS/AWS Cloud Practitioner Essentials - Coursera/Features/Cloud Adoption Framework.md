Cloud Adoption Framework

- 4 phase process to migrate to the cloud
- Envision phase
    
    - Focuses on demonstrating how cloud can accelerate the business outcome
- Align Phase
    
    - Focuses on identifying capability gaps across the six perspectives
- Launch Phase
    
    - Focuses on delivering pilot initiatives in production and demonstrating incremental business value
- Scale phase
    
    - Focuses on expanding production pilots and business value
